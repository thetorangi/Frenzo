# Frenzo

**Frenzo** is a modern, full-stack social networking platform inspired by Facebook — built with **Django** and **Vue.js**. It enables users to connect with friends, share updates, post photos, and interact through likes and comments.

This project showcases a powerful integration of a Python backend with a reactive JavaScript frontend, designed for real-time interactions and scalable social features.

## 🌟 Features

- User registration, login & profile management (Django auth)
- Post creation, editing, and deletion
- Like & comment system
- Friend requests and connections
- RESTful API built with Django REST Framework
- Responsive and dynamic UI with Vue.js

## 🛠️ Tech Stack

- **Backend**: Django, Django REST Framework
- **Frontend**: Vue.js, Vue Router, Vuex
- **Database**: SQLite3
- **Auth**: Django built-in authentication
- **APIs**: RESTful APIs between Vue & Django
